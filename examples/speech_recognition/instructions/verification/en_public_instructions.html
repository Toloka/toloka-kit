Each task contains a text and a audio record. You need to check if that pair is correct.

The pair is correct if:

  • The record content corresponds to text content.
  • The record does not contain grammatical or pronunciation errors (names that differ in pronunciation from a common noun are considered correct pronunciation).
  • The text does not contain sentences, which are not recorded.
  • The record sound is clear enough, there are no extraneous noise.

All others pairs are incorrect.
